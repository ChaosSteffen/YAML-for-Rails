h1. YAML for Rails

YAML for Rails provides a set of helper methods to deal with the great CSS-Framwork YAML (http://www.yaml.de).


h2. Installation

run in your apps root directory

	git clone git@github.com:Santino/YAML-for-Rails.git vendor/plugins/yaml_for_rails
